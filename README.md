## 🚀 Why Obistart?

Most browser start pages are cluttered with ads, trending articles, and distractions. **Obistart** replaces all that noise with a calm, intentional workspace that keeps you focused on what actually matters — your goals for the day.

> **No sign-up. No tracking. No distractions.** Just open a new tab and get to work.

## ⚡ Getting Started

### Prerequisites

- **Node.js** 18.17 or later
- **npm** (or your preferred package manager)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/obistart.git
cd obistart

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open **[http://localhost:3000](http://localhost:3000)** and set it as your browser's new tab page!

---

## 📖 Usage

### First Visit
Enter your name for a personalized greeting — stored locally, never sent anywhere.

### Daily Focus
Type your main focus and press Enter. Add subtasks with the **+** button and check them off as you go.

### Top Stories
Browse curated tech news from Hacker News and dev.to without leaving your start page.

### Pomodoro Timer
Start a focus session with customizable durations. An audio notification signals when your session is complete.

### Sound Player
Pick from ambient sounds (rain, café, nature) or radio streams to create your ideal work environment.

## 🔐 Privacy

Obistart is built with privacy as a core principle:

- **Zero tracking** — No analytics, no cookies, no fingerprinting
- **Local storage only** — Your name, focus, and preferences stay in your browser
- **No accounts** — Nothing to sign up for, nothing to manage
- **Minimal network** — Only fetches news feeds from public APIs
